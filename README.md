# Ocean View Resort - Online Room Reservation System

## Project Overview
This project is developed as part of CIS6003 Advanced Programming module.

The system is a web-based Online Room Reservation System built using Java EE (Servlets and JSP) following clean architectural principles and SOLID design practices.

## Technology Stack
- Java (Jakarta Servlet/JSP)
- Apache Tomcat
- Maven (WAR packaging)
- MySQL Database
- IntelliJ IDEA 2025

## Architecture
The system follows:
- MVC Pattern
- Layered (3-Tier) Architecture
- Singleton Pattern for Database Connection

## Branch Strategy
The project uses structured branch workflow:
- development
- qa
- uat
- master

Development → QA → UAT → Production
